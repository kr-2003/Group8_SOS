## Installation

- Meetify requires [Node.js](https://nodejs.org/) v10+ to run. 
<br>

- Go to the Meetify root folder.
```
cd Meetify
```
- Install the dependencies and devDependencies.
```
npm install
```

- Start the development server
```
npm start
```

- Now go to `http://localhost:3000`
<br>
